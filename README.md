# Cxp(Cuentas por pagar).
El paquete CXP (Cuentas por pagar) de V12 es el módulo encargado de la gestión de cuentas por pagar a proveedores y los documentos relacionados, funcionando como el núcleo para controlar las obligaciones financieras de la empresa. Su objetivo es centralizar la administración de facturas, pagos y saldos pendientes, garantizando trazabilidad y coherencia contable dentro del ERP.
